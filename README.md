# DigvijayCart: E-Commerce Website 🛒✨

A premium, highly responsive E-Commerce platform built entirely with modern HTML5, CSS3, and Vanilla JavaScript. 

## 🌟 Key Features
- **Glassmorphism UI**: Stunning translucent design language paired with a deep `#0f172a` dark mode theme.
- **Smart Navigation**: A dynamic, scroll-sensitive sticky header that shrinks and expands gracefully based on your scrolling behavior.
- **Intelligent Search Engine**: Real-time auto-suggest search bar that filters through products instantly without reloading the page.
- **Dynamic Cart System**: Fully functional, persistent shopping cart powered by `localStorage` (your items stay even if you close the browser!).
- **App-Like Mobile Experience**: Transforms on mobile devices to include a fixed bottom navigation bar, perfectly mimicking native apps like Flipkart or Amazon.
- **Premium Micro-interactions**: Hover effects, Quick View modals, Skeleton Shimmer loaders, and Toast Notifications.

## 🚀 Built With
- Pure HTML5
- Vanilla CSS3 (Custom grids, flexbox, animations)
- Vanilla JavaScript (No heavy frameworks or libraries)
